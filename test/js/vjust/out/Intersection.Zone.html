Intersection. Zone

new Zone(zone_ID)

A class that contains all data relevant to an intersectional Zone. Every Zone object, when initialized, creates four instances of Direction objects, one for each cardinal direction.
Parameters:
Name Type Description
zone_ID int The identification number for this zone. This number should always be between 1 and 6.
