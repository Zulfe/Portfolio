ModalFactory

ModalFactory

new ModalFactory(The, The, An, The, The, The)

Source:
  • classes.js, line 553
This is a class that helps with the creation of help modals on the application view. This tool will automatically store new modals in the help_modals div. An image or youtube video may be shown on the modal, but not both simultaneously. One ModalFactory object can be used to generate any number of modal elements on the HTML page. Simply modify the class's member variables and regenerate with ModalObject.injectModalHTML(ModalObject.generateModalHTML())
Parameters:
Name Type Description
The element_id element ID to be used in the creation of the modal
The header header text that is to be shown on the modal
An image image to be shown on the help modal. If an image shouldn't be used, specify "null"
The image_size size of the image following Semantic-UI descriptors (small, medium, large, or huge). If an image wasn't used, specify "null"
The youtube_import URL of a YouTube video to be added to the modal. If a YouTube video shouldn't be used, specify "null"
The description description text to be shown in the description of the modal
