DetailedVolume

new DetailedVolume(direction, left, through, right)

Source:
  • classes.js, line 1111
See:
  • BoundedDetailedVolume for the same organization but with direction described in terms of bounds ("northbound" being the southern branch).
A class for containing and organizing information relative to intersection branch movement volumes in an unbounded manner. In the unbounded case, a "north" DetailedVolume object describes vehicles entering and leaving via the northern branch of the intersection (southbound entering the intersection and northbound exiting the intersection).
Parameters:
Name Type Description
direction string The cardinal direction of the branch relative to the intersection point
left int The volume of vehicles traveling through the left turning lane of this branch
through int The volume of vehicles traveling through the through movement lane of this branch
right int The volume of vehicles traveling through the right turning lane of this branch

getPassengerCarEquivalent(left_truck_per, through_truck_per, right_truck_per) → {DetailedVolume}

Source:
  • classes.js, line 1158
Given the truck percentages for each movement, calculate the Passenger Car Equivalent (PCE) assuming that one truck equals two passenger cars. Return a new DetailedVolume object with the modified values. To perform this calculation, the number of trucks is added to the existing volume.
Example
// Convert an existing DetailedVolume object into a PCE-Enabled DetailedVolume object based on the original object's values
myDetailedVolume.getLeft() // → 500
myDetailedVolume = myDetailedVolume.getPassengerCarEquivalent(0.05, 0.10, 0.15);
myDetailedVolume.getLeft() // → 525
Parameters:
Name Type Description
left_truck_per float The percentage of vehicles in the left turn lane that are trucks
through_truck_per float The percentage of vehicles in the through lane that are trucks
right_truck_per float The percentage of vehicles in the right turn lane that are trucks
Returns:
A new DetailedVolume object with updated lane volumes
